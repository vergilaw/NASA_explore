Về NASA Space Explorer

Mục đích dự án

NASA Space Explorer là một ứng dụng web được xây dựng bằng Flask Python, tận dụng các API mã nguồn mở của NASA để mang đến trải nghiệm tương tác về khám phá vũ trụ. Dự án này được tạo ra nhằm mục đích giáo dục, cho phép người dùng truy cập và khám phá dữ liệu không gian từ NASA một cách thân thiện.

Các API được sử dụng

  • APOD (Astronomy Picture of the Day): Hiển thị hình ảnh thiên văn độc đáo mỗi ngày kèm theo mô tả.
  • Mars Rover Photos: Xem hình ảnh từ các tàu thăm dò Curiosity, Opportunity, Spirit và Perseverance.
  • Earth Imagery: Xem hình ảnh vệ tinh của Trái Đất tại các tọa độ cụ thể.
  • Asteroid NeoWs: Theo dõi các tiểu hành tinh gần Trái Đất và thông tin về mức độ nguy hiểm.
  • NASA Image and Video Library: Tìm kiếm trong kho lưu trữ phong phú các hình ảnh và video không gian.
  • DONKI (Space Weather): Cung cấp dữ liệu về các hiện tượng thời tiết không gian.

Công nghệ sử dụng

  • Backend: Python, Flask, Requests
  • Frontend: HTML5, CSS3, JavaScript, Bootstrap 5
  • APIs: NASA Open APIs

Hướng dẫn sử dụng

Để sử dụng đầy đủ các tính năng của ứng dụng, bạn nên đăng ký API key của riêng mình từ NASA API Portal. API key mặc định (DEMO_KEY) có giới hạn số lượng yêu cầu, có thể gây ra lỗi "rate limit exceeded" nếu sử dụng quá mức.

Sau khi có API key, bạn có thể thêm nó vào tệp .env trong thư mục gốc của dự án.

Hướng dẫn lập trình với NASA API

Dưới đây là một số ví dụ về cách sử dụng NASA API trong các ứng dụng của bạn:

Ví dụ Python đơn giản:

import requests

# Lấy Astronomy Picture of the Day
api_key = "YOUR_API_KEY"
url = f"https://api.nasa.gov/planetary/apod?api_key={api_key}"
    
response = requests.get(url)
data = response.json()
    
print(f"Title: {data['title']}")
print(f"Date: {data['date']}")
print(f"Explanation: {data['explanation']}")
print(f"Image URL: {data['url']}")
Ví dụ JavaScript đơn giản:

// Lấy dữ liệu về tiểu hành tinh gần Trái Đất
const API_KEY = "YOUR_API_KEY";
const startDate = "2023-01-01";
const endDate = "2023-01-07";
const url = `https://api.nasa.gov/neo/rest/v1/feed?start_date=${startDate}&end_date=${endDate}&api_key=${API_KEY}`;

fetch(url)
    .then(response => response.json())
    .then(data => {
        console.log(`Total asteroids: ${data.element_count}`);
        
        // Hiển thị số lượng tiểu hành tinh mỗi ngày
        Object.keys(data.near_earth_objects).forEach(date => {
            const asteroids = data.near_earth_objects[date];
            console.log(`${date}: ${asteroids.length} asteroids`);
        });
    })
    .catch(error => console.error('Error fetching data:', error));
